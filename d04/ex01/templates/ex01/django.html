Ex01 : Django, framework web.

Django History

Django was created in 2003 by a web development team at Lawrence Journal-World newspaper.

It was released publicly as open-source in 2005.

Django has since grown to become one of the most popular web frameworks for building web applications in Python.

Django Features

  • Fast development
  • Secure
  • Scalable

Django Installation

To install Django, you can use pip, the Python package manager. Run the following command:

pip install django

Starting a Project

To start a new Django project, use the following command:

django-admin startproject projectname

This will create a new directory called projectname with the necessary files and folder structure for your Django project.

Creating an App

To create a new app within your Django project, use the following command:

python manage.py startapp appname

This will create a new directory called appname with the necessary files and folder structure for your Django app.
