Ex01 : Moteur de template Django

Avec Django, les templates permettent de fabriquer des pages HTML dynamiques. C'est un moyen simple de mélanger du HTML avec des données qui viennent de ton code Python, comme des noms d'utilisateurs ou des listes d'articles.

1. Afficher des variables

Pour afficher une info qui vient de ta vue, tu l'entoures simplement de doubles accolades: {{ une_variable }}. Django la remplacera par sa vraie valeur.

{% verbatim %} 

<!-- Affiche le nom de l'utilisateur -->
<p>Salut, {{ user_name }} !</p>
{% endverbatim %}

2. Les Blocs {% block ... %}

Les blocs, c'est comme des trous à remplir. Tu crées un template principal (base.html) avec des zones vides définies par {% block nom_du_bloc %}. Ensuite, dans tes autres pages, tu utilises {% extends 'base.html' %} et tu remplis juste les blocs dont tu as besoin.

{% verbatim %} 

<!-- Fichier : base.html -->
<html>
<head>
    <title>{% block title %}Titre par défaut{% endblock %}</title>
</head>
<body>
    {% block content %}{% endblock %}
</body>
</html>

<!-- Fichier : page_enfant.html -->
{% extends 'base.html' %}

{% block title %}Un titre cool pour ma page{% endblock %}

{% block content %}
    <h1>Le contenu de ma page !</h1>
{% endblock %}
{% endverbatim %}

3. Les Boucles {% for ... in %}

Pour afficher chaque élément d'une liste, tu utilises une boucle for. Ça parcourt ta liste et répète un bout de HTML pour chaque item. N'oublie pas de fermer avec {% endfor %}.

{% verbatim %} 

<ul>
    {% for fruit in liste_de_fruits %}
        <li>{{ fruit }}</li>
    {% endfor %}
</ul>
{% endverbatim %}

4. Les Conditions {% if %}

Tu peux afficher des choses sous certaines conditions avec {% if ... %}. C'est utile pour montrer un message si l'utilisateur est connecté, par exemple. Tu peux aussi utiliser {% else %} et tu dois finir avec {% endif %}.

{% verbatim %} 

{% if user.is_authenticated %}
    <p>Bienvenue, {{ user.username }} !</p>
{% else %}
    <p>Salut l'ami, tu devrais te connecter.</p>
{% endif %}
{% endverbatim %} {% endblock %}
